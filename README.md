## ■ 制作背景・目的

AIビジネスに興味はあるものの、
「何から始めればよいか分からない」という初心者層に向けた
セミナー集客用LPとして制作しました。

### ▼ 実装・工夫した点

・ファビコン画像はChatGPTで生成  
・メイン画像横に「今なら！」「参加者特典あり！申し込みはこちら」を配置し、視線誘導を設計  
・メイン画像内に「期間限定」と表示し、緊急性を演出  
・「こちら」をクリックすると氏名入力欄へスムーズスクロール（離脱防止設計）  
・スクロール誘導アニメーション（縦線＋Scroll表示）を実装  
・特典訴求テキストにマーカー風装飾を追加  
・meta descriptionを設定し、SEO対策を実施  

### ▼ フォーム機能（JavaScript）

・氏名とメールアドレスの両方が未入力の場合はエラー表示  
・両方入力後にボタン押下で表示テキストと色を変更  
・登録完了を視覚的に分かりやすく演出  

※実際の送信処理（バックエンド連携）は未実装です。

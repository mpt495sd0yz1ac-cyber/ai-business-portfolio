#AIビジネス活用術セミナー　Webページ
■プレビューURL
AIビジネス活用術セミナーのLPページですHTMLとCSS、JavaScriptで構成し、メイン画像やキャプション入力フォームを含みます
[AIビジネス活用術セミナー_LP]https://mpt495sd0yz1ac-cyber.github.io/ai-business-portfolio/

■制作背景・目的
・ファビコンの画像はChatGPTで生成しました。
・テーマにあったように、メイン画像の横に「今なら！」「参加者特典あり！申し込みはこちら」と実装し、メイン画像の中に「期間限定」と記載することで、早期の申し込みを促しています。
・「こちら」をクリックすることで、氏名の入力欄まで飛ぶことでスクロールせずに申し込みできるようにしました。(離脱を防ぐため)
・スクロール誘導用のアニメーションを追加（縦線＋Scroll表示）
・テーブルの下に、「今申し込みいただいた方に参加者特典プレゼント!」の文章に黄色のマーカーを引き、早期の申し込みを促すようにしています。
・SEO対策として、「本ページは、AIをビジネスに導入したいが「何から始めればいいか分からない」という方に向けて、初心者でも失敗しないAI活用方法を具体例付きで解説しています。」を記載しています。
・氏名とメールアドレスを両方入力しないと申し込みボタンを実行できないように実装しています。(両方入力しないと「氏名とメールアドレスを入力してください」と表示しています)
・氏名とメールアドレスを両方入力してボタンを押すと、「参加者特典GET！」の文章に切り替わり、ボタンの色を変えることで、登録が完了したことを視覚的に分かるようにしました。(氏名・メールアドレスに「あ」などの文字を入力すれば実行されます)
・「参加者特典つき！今すぐ登録する」のボタンをクリックすると参加者特典GET！とボタンの色に切り替わります。(JavaScriptを実装しています)
・今回はPCサイズ、タブレット・スマホサイズで制作しています。
